# Pinas

This gem provides model for Philippines location

## Installation

Add this line to your application's Gemfile:

    gem 'pinas'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install pinas

## Usage

    $ rails g pinas:install

    $ rake db:migrate

    $ rake pinas:load_data
